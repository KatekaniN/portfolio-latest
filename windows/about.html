Professional Overview

Hey there! I'm Katekani, a developer who loves turning ideas into interactive web experiences people can actually click, drag, and play with. This whole portfolio is a Windows 11 desktop I built from scratch in vanilla JavaScript, so go ahead and open a few windows.

My foundation is strong JavaScript fundamentals, and I build with React, HTML, CSS and responsive design, wiring everything up to REST APIs. I started out studying Mathematical Statistics at the University of Pretoria, taught myself to code, and now build for real brands as a developer at Ogilvy South Africa. I'm curious by nature, I take ownership of what I build, and I'm always learning, currently Java and how to build with LLMs.
